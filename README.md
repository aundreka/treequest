# TreeQuest

## Overview

TreeQuest is an educational web-based game designed to help students understand **tree traversals** in a fun and interactive way. It was created as a learning aid for classmates to better grasp traversal concepts through gameplay rather than traditional lectures.

The game focuses on reinforcing key traversal techniques by allowing players to actively explore and interact with tree structures.

---

## Features

* Interactive tree visualization
* Practice for common tree traversal methods:

  * Preorder Traversal
  * Inorder Traversal
  * Postorder Traversal
* Game-based learning approach
* Immediate feedback on answers
* Simple and intuitive user interface

---

## Tech Stack

This project was built using:

* HTML
* CSS
* JavaScript

No external frameworks or libraries were used.

---

## How to Run

1. Download or clone the repository
2. Open the project folder
3. Run the game by opening the `index.html` file in any web browser
